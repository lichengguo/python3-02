循环语句可以提取每一个复合类型数据的成员

列表

{% for i in list1 %}
  • {{ i }}
    • {% endfor %}

    元组

    {% for foo in tuple %}
  • {{ foo }}
    • {% endfor %}

    字典

    {% for i,k in dict1.items %}
  • {{ i }} - {{ k }}
    • {% endfor %} {% for foo in dict1 %}
  • {{ foo }} - {{ dict1.foo }}
    • {% endfor %}

    输出下标

    {% for foo in list1 %}
  • 序号{{ forloop.counter0 }} - {{ foo }}
    • {% endfor %}
